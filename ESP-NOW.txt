ESP-NOW protokol komunikasi nirkabel yang dikembangkan oleh Espressif Systems, yang dirancang khusus untuk mikrokontroler ESP8266 dan ESP32. Protokol ini memungkinkan komunikasi cepat dan efisien antara beberapa perangkat tanpa perlu terhubung ke jaringan Wi-Fi.
Komunikasi Langsung: ESP-NOW memungkinkan komunikasi langsung antara perangkat ESP8266 dan ESP32 tanpa perlu terhubung ke router atau titik akses Wi-Fi. Ini memungkinkan pertukaran data yang cepat dan efisien dalam jarak yang relatif dekat.
Koneksi Tanpa Pengamanan: Protokol ini sering digunakan dalam aplikasi di mana kecepatan dan efisiensi lebih penting daripada keamanan. ESP-NOW tidak memiliki mekanisme otentikasi atau enkripsi bawaan, sehingga penting untuk diimplementasikan di atas lapisan keamanan tambahan jika dibutuhkan.
Broadcast dan Unicast: ESP-NOW mendukung mode komunikasi broadcast dan unicast. Mode unicast digunakan untuk mengirimkan data ke perangkat tertentu, sedangkan mode broadcast memungkinkan pengiriman data ke semua perangkat di jaringan ESP-NOW.
Pemrograman Mudah: Penggunaan ESP-NOW relatif mudah dalam pengembangan perangkat IoT karena ada pustaka perangkat lunak yang tersedia untuk mikrokontroler ESP8266 dan ESP32. Pustaka ini menyediakan API yang sederhana untuk menginisiasi, mengirim, dan menerima data melalui ESP-NOW.
Konsumsi Daya Rendah: ESP-NOW dirancang dengan memperhatikan konsumsi daya, sehingga cocok untuk aplikasi IoT yang membutuhkan efisiensi energi. Ini memungkinkan perangkat ESP8266 dan ESP32 untuk beroperasi dalam mode hemat daya saat tidak mengirim atau menerima data.
